## Usage

```bash
$ nuxtron init my-app --example with-javascript-vuetify

$ cd my-app

# Install dependencies
$ yarn (or `npm install`)

# Run development mode
$ yarn dev (or `npm run dev`)

# Build packages
$ yarn build (or `npm run build`)
```
